• More information about this page

This interactive web page was developed as an assignment for the course Responsive Website Basics:

  • It makes use of the Bootstrap framework to accomplish responsiveness to page resizing. Resize your browser window to see the Bootstrap grid in action.
  • It makes use of JavaScript to accomplish interactivity:
    • Click the single item in the navigation bar at the top to display the description of this website.
      Click it again to hide the description.
    • Click the text or the buttons in the central image to swap image galleries about the topics in and out of the bottom area of the page.
      The central image and its components are created with a Bootstrap Jumbotron component.
  • HTML, h1 and img tags are used multiple times. All parts of the design are styled with the CSS in the main.css file.
    Note on font: I'm loading the Lato font from the online Google font repository, but a fallback font is defined in case of problems.
